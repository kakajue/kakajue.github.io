组装计算机

电脑分类：台式机（一体式电脑，分体式电脑），平板电脑，笔记本

1.电脑办公

2.网上冲浪

3.管理文件

4.试听播放

5.游戏娱乐

电脑游戏分为：1.网络游戏，2.单机游戏（游戏光盘安装后），3.网页游戏

电脑硬件：主要硬件设备，常用外部设备

主要硬件设备：

1.主板：机箱里，矩形电路板，BIOS芯片，I/O控制芯片，面板控制开关

开放式结构，6-15扩展槽，控制卡插接，局部升级子系统，４－８个USB接口（U盘，打印机，扫描仪，数码相机，鼠标），一个PS／２键盘和鼠标接口，１－２个网络接口，一个１３９４接口（数码摄像机等设备），一个eSTATe接口（连接STAT硬盘），多个音频接口

2.CPU：电脑的“大脑”，控制操作系统，在主板的CPU插座上，有运算器，控制器，寄存器。运作原理：提取-解码-执行-写回

CPU从存储器，或者高速缓冲存储器中取出指令，放入寄存器中，译码指令，执行指令。

对CPU的编程——>电脑的可编程性

3.内存：在主板的内存插槽里，作用：暂时存放数据（CPU的和硬盘的），使电脑稳定运行。

4.硬盘：在机箱上的驱动器架内，通过数据线与主板相连。主要的存储媒介。由铝制或者玻璃制的碟片组成，碟片外有铁磁性材料。（PS：硬盘是一个一个盘片叠加的，每个盘面被分成相同的磁道，并从外缘的“0”开始编号，相同编号的磁道形成一个圆柱，称为磁道的柱面）STAT接口和IDE接口

5.显卡：在电脑主板的PCI Express（或者AGP，PCI）插槽中。控制显示器的正确显示。

6.机箱：就是一个外壳。

7.显示器：

8.电源：机箱的电源架上，把交流电换成直流电供配置使用。转换频率在70%-80%

9.键盘和鼠标：使用PS/2或者USB接口和主机相连

10.光驱：读光盘的。CD-ROM驱动器：只读CD； DVD光驱：可以读CD也可以读DVD；刻录机：写入。常用外部设备：

１１.数据线：STAT硬盘数据线，电源线，IDE数据线，

1.打印机

2.摄像头

3.移动存储设备

4.耳机与耳麦

5.麦克风

6.音箱

电脑的软件

系统软件：

1.操作系统软件：是介于软件和硬件之间的一个结构层。功能模块：CPU管理，存储管理，设备管理，文件管理。

2.语言处理软件：机器语言；汇编语言；高级语言。

3.系统服务程序：是随着操作系统一起安装的，并且作为操作系统的一部分提供单机或者网络服务。（任务管理器）

4.数据库管理系统：数据库的定义是以一定方式存储起来的，具有相关性的数据的集合。管理系统就是对数据库进行建立，管理，维护，使用。数据库管理系统软件：Access，Oracle，SQLServer,DB2。

应用软件：

1.应用软件包：Office等

2.用户程序：自己需要的软件

某大佬督促我这个菜鸟来写一下东西？又是被完虐的一天？

————手动分割————

1.书里说Spring可以解决Java EE开发的所有问题 因为它提供了Ioc容器，AOP，数据访问，Web开发，消息，测试等等（以上专业术语一个都不懂，只是单纯的码下来）

2.Spring使用简单的POJO（Plain Old Java Object 无限制的普通Java对象），来进行企业级开发。每一个被Spring管理的Java对象都被称之为Bean，而Spring提供了一个IoC容器用来初始化对象，解决对象间的依赖管理和对象的使用（同上，码着先，感觉有点用？）

3.Spring是模块化的，每一个模块至少对应一个jar包？（所以jar包又是什么？😫）

4.项目构建工具：Maven，Ant，Gradle

5.用Maven管理项目，项目依赖的jar包不包含在项目内，而放在用户目录下的.m2文件夹下

6.

今天下载maven的时候还出现了问题，找不到要下载哪一个？

下载网址：https://maven.apache.org/download.cgi

7.添加环境变量的问题

电脑——属性——高级系统设置——环境变量

配置M2_HOME D:\Spring Boot\apache-maven-3.5.3

Path %M2_HOME%\bin;D:\Spring Boot\apache-maven-3.5.3\bin;（两个随便写一个就好）

cmd mvn -v

8.Maven 基于项目对象模型的概念运作的，每一个Maven都有一个pom.xml来管理项目依赖和项目编译

9.Maven的运作方式：根据dependency中的依赖配置，直接通过互联网，在Maven中心库下载相关的依赖包到.m2目录下。

关于依赖jar包的dependency，参考网址 https://mvnrepository.com

但有些jar包在Maven中心库没有，如Oracle，需要安装Oracle驱动到本地库：

mvp install:install-file -Dgroupld=com.oracle “-DartifactID=ojdbc14”

“-Dversion=10.2.0.2.0” “-Dpackaging=jar” “-Dfile=D:\ojdbc14.jar”

10.下载Spring Tool Suite https://spring.io/tools/sts/all

如何搭建个人博客（Hexo+Github）

先下nodejs ，选择Windows.msi 64位

淘宝源：https://blog.csdn.net/quuqu/article/details/64121812

持久使用

win + r +cmd +npm config set registry https://registry.npm.taobao.org

然后执行 npm install -g hexo

在D盘新建一个目录（文件夹）ajue

hexo初始化 win + r +cmd +d: +cd ajue +hexo init

执行 hexo start

执行 hexo s或者hexo server

打开浏览器 访问 http://localhost:4000

hexo init创建项目

hexo new 2018-5-6创建新页面

hexo server启动服务

下载 git

登陆GitHub 并且在上面创建一个项目 Start a project kakajue.github.io

本地找到 D盘>ajue>_config.yml 打开方式 VS code

拖到最后面修改一下 ：

deploy：

​ type : git

​ repository: 如下图找那个网址那个绿色的地方下拉

​ branch: master

记得保存啊兄dei ！！！

执行 npm install hexo-deployer-git –save

执行 hexo d 然后你会看到一下界面

然后登陆你的GitHub

把那个红框里面的去掉keep your adress private

hexo g(hexo generate )生产，生成

hexo d(hexo deploy )部署
